Home

Drafter Workspace

As everything is either a Card or an Entry, glance over where their actions are. Then, take a look at the Cards and compare them with the next section.

Previewer Workspace

In Previewer, Cards become interactive pages. In this workspace, you can change the page background among other settings.

Initial Page

Previewer opens at the selected Card. Otherwise, at the topmost.


Initial Workspace

Files remember their last active workspace.

Undo works in Previewer too

Both workspaces share their undo history. For example, changing an Entry value in Previewer modifies it in Drafter as well. If that's undesirable, you can either:

  • File → Duplicate before previewing, or
  • Previewer → Preview in New Tab. This way Drafter and Previewer will have independent undo histories. But you'll need to make the final edit in the tab you want to be saved in your browser storage. That edit could be arbitrary, or an Undo + Redo.

Tips

Exporting for Graphic Design Programs

In Previewer, use the browser's Print → Save as PDF. That PDF serves as an editable starting point you can import into Adobe® Illustrator or Inkscape®.

Hiding Toolbars

You can hide the Toolbar and Side Panels using the View menu, because most actions are available in context menus (right-click) and Shortcuts, and the top Menubar has all them.
